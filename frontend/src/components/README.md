# StartPage Component

This folder contains the main landing page component and its styles for the Tairis project.

## Files
- `startpage.jsx`: The React component for the landing page.
- `startpage.css`: Styles for the landing page component.

## Usage
Import and use the `StartPage` component in your main app entry point:

```jsx
import StartPage from './components/startpage.jsx';

function App() {
  return <StartPage />;
}
```

## Customization
- Edit `startpage.jsx` to change the landing page content or structure.
- Edit `startpage.css` to update the styles.

---

This component is part of the Tairis project, a Doctor-Free First Aid Decision Tree web app.
